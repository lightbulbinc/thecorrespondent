Create a media profile

Use this page type to create media profiles. Then attach these profiles to your pages (content or aggregate) to display the updatable media on those pages.

The benefit of this approach is that when you update (and publish) a media profile, the updates will be immediately visible in the host pages.

First, start off with deciding the orientation (vertical or horizontal) of the media group. This is how the media will display in the published output.
